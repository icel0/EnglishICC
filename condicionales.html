ZERO CONDITIONAL

Se usa para hechos generales, verdades universales, reglas.

Estructura:

If + Present simple , Present simple.

Ejemplos:
  • If you heat water to 100°C, it boils.
  • If don´t sleep, I feel tired.

FIRST CONDITIONAL

Se usa para situaciones reales o posibles en el futuro.

Estructura:

If + Present simple , Will/ Won´t + Verbo en forma base + Complemento.

Ejemplos:
  • If it rains tomorrow, we will stay home.
  • If I don´t study, I won´t pass the exam.

SECOND CONDITIONAL

Se usa para situaciones hipotéticas o improbables en el presente o futuro.

Estructura:

If + Past simple , Would/Wouldn´t + Verbo en forma base + Complemento.

Ejemplos:
  • If you were here, we would laugh about their vacant stares.
  • And if I didn´t know better, I would think you were talking to me now.

THIRD CONDITIONAL

Se usa para situaciones hipotéticas o improbables en el presente o futuro.

Estructura:

If + Past perfect , Would/Wouldn´t have + Verbo en pasado participio + Complemento.

Ejemplos:
  • If they haf left earlier, they would have caught the train.
  • If you had loved me, you would never have hurt me.

MIXED CONDITIONAL

Se usa para situaciones hipotéticas o improbables en el presente o futuro.

Estructura:

Pasado ---> Presente

Presente ---> Pasado

  • If you didn´t mean what you wrote in that song about me.
  • If I was stronger, I would have called you out.
